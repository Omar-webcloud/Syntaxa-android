# Syntaxa for android





Syntaxa is an Android learning app built with Flutter to help users improve their English through simple and interactive quizzes. The app focuses on making grammar practice engaging, smooth, and easy to use with a clean interface.

## Features

Daily quizzes to build grammar skills  
Built in dictionary with meanings and pronunciation  
Practice modes for sentence correction and structure  
Progress tracking to monitor improvement  
Gamification with rewards and milestones  
Light and dark theme support  

## Tech Stack

Flutter and Dart  
Material UI for Android  
State management using Provider or similar  
API integration for data  

## Getting Started

Clone the repository  
Run flutter pub get  
Run flutter run on an Android device or emulator  

## Platform

Android only  

## Summary

A simple and well structured Flutter app focused on performance, usability, and effective learning.
